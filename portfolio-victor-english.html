Professional Projects

All of these projects were completed during my work-study program as well as during my years as a freelancer.

Dachser Legal Text

Legal Text is an application that manages the company's legal documents. A typical use case is when a board member retires: all documents referencing that person must be updated. To address this need, I built a web service capable of updating texts directly in the database.
Technologies used:
Figma for graphic design
Angular on the Frontend
Spring Boot on the Backend
IBM DB2 for the Database

Dachser Web Community

Another mission I was assigned was to work for the Dachser Web Community, where I created reusable components to save time for other teams.
This is what taught me to write cleaner, more modular code.

Quiz Mashup for the Archilan Event

Having already worked with real-time applications (via WebSocket), I designed for the Archilan Association a Kahoot-style application to host Quiz events on Mashups.
The project was a success, bringing together 30 players in real time via WebSocket.

Lawyer RAG Chatbot

Development of an advanced legal RAG chatbot that answers any question on criminal law. Technologies used: Flask with Python for the backend architecture, the OpenAI API for language analysis, and Milvus for data indexing (vector database).

Kahoot / Mentimeter WebApp

Development of a webapp similar to Kahoot / Mentimeter, using WebSockets for real-time communication, Node for the backend, React for the user interface, and Firestore NoSQL for data management.
A project that taught me a great deal:
- Two-way real-time communication via WebSockets.
- Various asynchronous API calls (Firebase, Firestore, OpenAI).
- Cookie management.
- Shadcn as the main component library.
- AI to allow users to create their own forms with a simple prompt.

Tag Direct

I implemented a mechanism to display differentiated pricing and products. B2B customers see certain products at specific prices, while B2C customers access different pricing.
