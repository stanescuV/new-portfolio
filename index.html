Tag Direct

J’ai mis en place un mécanisme permettant d’afficher des tarifs / produits différenciés. Les clients B2B voient certains produits à des prix spécifiques, tandis que les clients B2C accèdent à d’autres tarifs.

Dachser Legal Text

Legal Text est une application qui gère les textes légaux de l'entreprise. Un cas d'usage classique est le départ à la retraite d'un membre du board : tous les documents où cette personne est citée doivent être modifiés. Pour répondre à ce besoin, j'ai construit un webservice capable de mettre à jour les textes directement en base de données.
Pour réaliser ce projet, j'ai utilisé :
Figma pour la conception graphique
Angular côté Frontend
Spring-Boot côté Backend
IBM DB2 côté Base de Données
